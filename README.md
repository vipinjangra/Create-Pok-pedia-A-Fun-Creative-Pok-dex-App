# Poképedia - Flutter Pokédex App

This app displays Pokémon using PokéAPI.

## Features
- Login screen with Lottie animation
- Pokémon list fetched from API
- Grid UI with cards
- Pokémon images

## API Used
https://pokeapi.co/

## Packages
- http
- lottie
- google_fonts
